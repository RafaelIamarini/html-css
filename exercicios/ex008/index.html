PRINCIPAIS FORMATAÇÕES

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <B> (nao semântica).

Já nesta frase temos um termo em destaque usando a tag <STRONG>(semântico).

Itálico / Ênfase

Nesta frase temos um termo Itálico usando a tag <I> (nao semântico).

Nesta frase temos um termo em Ênfase usando a tag <EM> (Semântico).

Texto Marcado

Nesta frase temos um Marcador de Texto usando a tag <MARK>.

Texto Grande e Pequeno

Estamos criando um texto Grande e um texto pequeno neste parágrafo.

Texto Deletado

Podemos marcar textos Excluidos para indicar que ele seja lido, mas não considerado, usando a tag <DEL>.

Texto Inserido

Podemos marcar textos Inseridos para indicar que ele seja lido, e considerado, usando a tag <INS>.

Texto Sobrescrito

Neste formato usamos a tag <SUP>, para inserir coisas do tipo x20+3

Texto Subscrito

Neste formato usamos a tag <SUB>, para inserir coisas do tipo H2O
